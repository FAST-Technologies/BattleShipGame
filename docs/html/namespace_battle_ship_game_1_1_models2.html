 Класс SoundManager предоставляет методы для воспроизведения звуковых эффектов в игре "Морской бой". Звуки генерируются программно как синусоидальные тона и воспроизводятся через библиотеку NAudio. Все методы являются статическими и потокобезопасными благодаря асинхронному воспроизведению в отдельных задачах.
