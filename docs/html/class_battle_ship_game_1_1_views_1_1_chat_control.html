Detailed Description

Пользовательский контрол для отображения и управления чатом в сетевой игре. Обеспечивает отправку и получение сообщений, форматирование времени и управление UI чата.

Контрол используется только в сетевом режиме игры для общения между игроками. Автоматически подключается к ChatManager для обработки сетевых сообщений.
