\doxysubsection{Detailed Description}
Локатор представлений (View Locator) для паттерна MVVM. Преобразует View\+Model в соответствующий View на основе соглашения об именовании. 

По умолчанию ищет View с тем же именем, что и View\+Model, но с заменой "{}\+View\+Model"{} на "{}\+View"{}. Например\+: \doxylink{class_battle_ship_game_1_1_view_models_1_1_main_window_view_model}{Main\+Window\+View\+Model} → Main\+Window\+View
