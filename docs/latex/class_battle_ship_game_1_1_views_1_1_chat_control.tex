\doxysubsection{Detailed Description}
Пользовательский контрол для отображения и управления чатом в сетевой игре. Обеспечивает отправку и получение сообщений, форматирование времени и управление UI чата. 

Контрол используется только в сетевом режиме игры для общения между игроками. Автоматически подключается к \doxylink{class_battle_ship_game_1_1_networking_1_1_chat_manager}{Chat\+Manager} для обработки сетевых сообщений.
