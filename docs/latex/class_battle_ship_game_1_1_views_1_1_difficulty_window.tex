\doxysubsection{Detailed Description}
Окно выбора уровня сложности бота для режима игры против компьютера. Предоставляет выбор между тремя уровнями сложности\+: Легкий, Средний, Сложный. 

Используется только в режиме игры против компьютера (Game\+Mode.\+Vs\+Computer).
