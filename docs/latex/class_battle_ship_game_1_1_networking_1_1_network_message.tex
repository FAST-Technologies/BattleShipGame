\doxysubsection{Detailed Description}
Представляет сетевое сообщение, передаваемое между клиентом и сервером. 

Сообщение состоит из типа (команды) и набора ключ-\/значение пар данных. Формат сериализации\+: "{}\+Type\+:key1=value1;key2=value2"{}.
